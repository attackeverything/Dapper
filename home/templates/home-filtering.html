Filtering The Leaderboard

The leaderboard includes the ability to filter based on three different attributes.

These attributes are:

  • Author
  • Academic Affiliation
  • Model Type

Once a specific Author, Academic Affiliation, or Model Type is selected from the drop down menu below, only submissions meeting that criteria will be displayed.

As well, each filtering drop down menu contains a search bar at the top that allows filters to be found easily in the case that there are many different criteria.
