Submission Process

Below you will find the required steps to successfully submit a model to this system.

Step 1:

Create the SOC estimator function that you want to submit to be tested and ensure it is in the form of a .m or .mat file.

Step 2:

Navigate to the "Submit Model" page and using either drag and drop or selecting the file, upload the .m or .mat file. Create a name for the model in the "Model name" field and select the model type that you are uploading from the "Model Type" drop down menu. Lastly, decide if you would like your submission to be displayed on the leaderboard or not. An example is provided below of a complete submission.

Step 3:

Navigate to the "View Submissions" page and you will be able to check the status of your submission. Once the tests have been completed, this page will show all of the results of the submission includng helpful graphs to illustrate these results.

Step 4:

If during the submission process, the "public" option was chosen, check the leaderboard to see how you rank compared to others that have made a submission.
