Condensed vs Detailed Views

The leaderboard contains two viewing options and the details about these views can be found below:

Condensed View

This view only contains the columns related to the user that submitted the model and the basic model details. It is used to avoid the clutter of the full leaderboard.

Detailed View

This view includes all the columns of the leaderboard and will give a full overview of the results of each submission.
