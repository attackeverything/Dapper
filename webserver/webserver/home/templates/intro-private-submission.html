Public/Private Submissions

When submitting a model to the system, there are two options for how your submission will be displayed.

Public Submission

This is the default appearance of submissions and will result in the submission being displayed on the leaderboard along with the user's details.

Private Submission

This is an additional setting based on user preference that results in a submission not being displayed on the leaderboard. This submission and all of the stats assoicated with it will only be visible to the user that submitted it.

Toggle

When making a submission, you will notice a toggle that will allow you to chose between these two options.
