Sorting

The leaderboard page and the view submissions page contains the ability to sort by any of the columns and the sorting can be done either in ascending or descending order.

To sort the leaderboard or your submissions, simply select the column you would like to sort by from the drop down menu pictured below while on the leaderboard page and it will automatically be sorted.
